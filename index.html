FazMeRir.js, a jQuery plugin to show pretty prices in your site

This plugin is a graceful combination of Lettering.js, priceFormat.js and more than
2 predefined CSS styles
from the community.

How to use

1

After including jQuery 1.7.2+, download and include Lettering.js, priceFormat.js and FazMeRir.js in your project: 

<!-- jQuery -->
<script src="js/jquery.js"></script>

<!-- FazMeRir Dependencies -->         
<script src='js/jquery.price_format.js'></script>
<script src='js/jquery.lettering.js'></script>  

<!-- FazMeRir Plugin -->
<script src="js/jquery.fazmerir.js"></script>

2

Define your price elements: 

<strong class="price">1234567</strong>
* You don't need to specify the money format or cents separator here, it will be magicaly added after.

3

Select your element and call the fazMeRir() method: 

$( '.price' ).fazMeRir();

All Done!

This will be result in a sliced price element, each piece with contextual classes: 

<strong class="price fazmerir-demo">
    <span class="char1 prefix">R</span>
    <span class="char2 prefix">$</span>
    <span class="char3 prefix"> </span>
    <span class="char4 thousands">1</span>
    <span class="char5 thousands">2</span>
    <span class="char6 thousands separator">.</span>
    <span class="char7">3</span>
    <span class="char8">4</span>
    <span class="char9">5</span>
    <span class="char10 cents separator">,</span>
    <span class="char11 cents">6</span>
    <span class="char12 cents">7</span>
</strong>

Great! Now you have
individual characters control!

Now you can easely style your prices using the semantical classes, like this:

.price {
    color: goldenrod;
}
.price .prefix {
    font-size: 40%;    
    vertical-align: top;
}
.price .cents {
    font-size: 50%;    
    vertical-align: bottom;
}

1234567

...or you can use any of 2 predefined styles, just setting 'style' parameter with the class name of style, in arguments of method:

$( '.price' ).fazMeRir({
    'style' : 'serious',
});

OPTIONS

Parameter Type Options
stylespath: String
The absolute uri to the styles folder
style: String (optional)
The class (name) of predefined style.
The respective CSS file needs to be present in the styles path.

You can pass any parameter from priceFormat.js that it will be work like a charm.

Parameter Type Default
prefix: String 'US$ '
suffix: String
centsSeparator: String '.'
thousandsSeparator: String ','
limit: Boolean FALSE
centsLimit: Number 2
clearPrefix: Boolean FALSE
clearSufix: Boolean FALSE
allowNegative: Boolean FALSE
insertPlusSign: Boolean FALSE
clearOnEmpty: Boolean FALSE

Examples: 

$( '.price' ).fazMeRir({                
    'style': 'serious',
    'stylespath': 'js/styles/',
    'prefix': 'R$ ',
    'centsSeparator': ',',
    'thousandsSeparator': '.',
    'suffix': '/mês',
});
$( '.price' ).fazMeRir({                
    'style': 'golden',
    'stylespath': 'bower_components/fazmerir.js/styles/',
    'prefix': 'US$ ',
    'centsSeparator': '.',
    'thousandsSeparator': ',',    
});
